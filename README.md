# Tsui board  

入力内容を外部送信しない・広告なしで動作する、雑に絵を書けるWebホワイトボード。  

- 入力した内容は端末の外に出ない  
- 広告なし、トラッキングなし  

## 使い方  

1. [https://hajimetwi3.github.io/tsui-board/](https://hajimetwi3.github.io/tsui-board/) をブラウザから直接開く
